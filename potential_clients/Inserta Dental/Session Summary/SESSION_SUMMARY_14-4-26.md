## Methodology

**Data sources:**
- 60-day search terms report (64,908 rows, all 4 campaigns)
- Daily search terms report for 13 April 2026 (2,346 unique terms)

**Classification approach (final):** Default-block logic — a term is only kept running if it contains (a) a DBD service keyword (implant, veneer, Invisalign, composite bonding, All-on-4, All-on-6, cosmetic, etc.), (b) London/Greater London location, (c) clear buying-intent phrase — AND contains no wrong-intent signals (brand name, wrong location, abroad, NHS, cheap, denture, non-DBD procedure).

**Review process:**
- Each batch verified against the full 60-day data before presenting
- Every recommendation cross-checked for conversion history (no blocks on terms with legitimate conversion patterns without explicit user decision)
- Multiple passes to catch missed brand patterns, pre-implant procedures (sinus lift, bone graft), over-vague terms, and brand compound words ("X dental" pattern)

## What's preserved (kept running)

Explicitly kept — potential DBD clients:

- Service-specific variants (Hollywood smile, invisible aligners, same day teeth)
