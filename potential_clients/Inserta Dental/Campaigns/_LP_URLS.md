# Landing Page URLs — 70 ad groups

All LPs sit under `https://dentalbydesign.co.uk/google/`.

URL slug pattern:
- CORE = `{base}`
- COST (= COST + CHEAP merged) = `{base}-cost`
- LOCATION = `{base}-london`
- FINANCE = `{base}-finance`
- COMPARISON = `{base}-vs-{other}`
- INFO = `{base}-explained`
- BRAND = `{base}-{brand-name}`
- NHS = `{base}-nhs`

Default = one LP per sub-group (70 total). Consolidate during build only where messaging would be identical.
