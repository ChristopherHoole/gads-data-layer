M7 Wireframe — 4-Tab Recommendations UI

Applies to: recommendations.html (global /recommendations page) and campaigns.html (Campaigns → Recommendations tab). Card design matches existing M6 card layout exactly.

1. TAB BAR — 4 tabs replacing the existing Pending / Monitoring / History tabs

↑ Active tab: blue underline + blue text. Badge colours: default=Pending, amber=Monitoring, green=Successful, grey=Declined. Replaces existing "Pending / Monitoring / History" tabs.

Implementation note: Tab switching is pure JS — no page reload. Pending is default active. Badge counts computed client-side from card data grouped by status.
